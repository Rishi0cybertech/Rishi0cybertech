# 💫 About Me:
I am Rishi Gauttam, a 2nd year B.Tech Cybersecurity student at Swami Rama Himalayan University, Dehradun. I joined SRHU with one goal — to build things that actually matter.<br>In my first year itself, I co-invented my first patent — D-VISS (Patent No. 202611029490A), a decentralized vehicular safety system, and then Invented CellVeda (Patent No. 202611045822), an AI-adaptive battery desulfation system. CellVeda is now selected for the STPI Startup Program through SRHU's ACIC SIIC incubator, with funding approved and prototyping actively underway.<br>As an active member of the SRHU ACM Student Chapter, I have helped as a host and contributed to event management — helping organize and run technical events for the student community.<br>Beyond academics, I founded CellVeda, run a YouTube channel called Tech with Rishi, and maintain 500+ LinkedIn connections across the cybersecurity and startup ecosystem.<br>I build at the intersection of cybersecurity, IoT, and embedded systems — one patent, one tool, one problem at a time.<br>"Dreams are those which make you sleepless."
